void motor_rot_rel ( short  angle)

rotate the Pololu bot for arbitary angle

In timed rotation mode, This function essential wraps around internal_motor_rotate. If the robot is to rotate above 360 degrees, it rotate back to 0 degree first, then rotate to the resulting angle.

The robot can rotate back to the starting position quite accurately by rotating in the reverse direction for the same amount of time. However asking the robot to rotate 360 degrees does not result in accurate 360 degrees rotation.
