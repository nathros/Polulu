Detailed Description

Centralised Configuration File.

This file contains the configuration data for IR sensor, collision sensor, motor, and the mouse.

Note
For more details on pin configuration, please visit http://www-users.cs.york.ac.uk/~pcc/MCP/MbedPins.html.
Motor related definitions:
  • LEFT_BIAS and RIGHT_BIAS: these are multiplied to the speed of the repective motors.
For our robot (6):
  • define COLLISION_SENSOR_PORT 0
  • define COLLISION_SENSOR_PIN_NUM 16
  • define COLLISION_SENSOR_PIN (1 << COLLISION_SENSOR_PIN_NUM)
  • define LEFT_BIAS 1
  • define RIGHT_BIAS 0.85
