Detailed Description

mouse module source code

This module handles everything that's got to do with the mouse.

Note
Correction factors for the mouse:
  • SHIFT_T corrects for the bias in rotation,
  • SHIFT_L corrects for the bias in distance,
  • T_PER_DEGREE is the T reading from the mouse when the vehicle rotate about its centre for 180 degrees
  • L_PER_CM is the distance the R reading from the mouse when the vehicle moves in a straight line for 1 meter.
