Note
PID_FACTOR scales the pid value, so it is between (-2000 .. 0 .. 2000)
  • 5.555 -> (-180 .. 0 ..180)
  • 11.111 -> (-90 .. 0 .. 90)
  • 22.222 -> (-45 .. 0 .. 45)
