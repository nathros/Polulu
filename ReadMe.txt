This is a program for the m3pi Robot + mbed NXP LPC1768 microcontroller
see: https://www.pololu.com/product/2153

Directory/POLULU
This code runs on the micro-controller can follow black lines using base sensors.
Measure travelled distance with USB mouse sensor and follow walls with 4xIR sensor.

Directory/PoluluNav
Runs on a host computer connected to the micro-controller, user controls the robot
and the robot send back IR data. This is used to generate a real-time image of the
environment (see screenshots).
